# Payslip Generator (Next.js scaffold)

Minimal internal web tool scaffold for generating payslips from payroll uploads.

## Includes

- No auth, no database
- Upload `.xlsx`, `.xls`, `.csv`
- Required payroll columns parser
- Fixed payslip preview component
- Core calculation utilities
- PNG export + ZIP download scaffold

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
